# L3150 Jatinegara (PJ, PQ)

## Paper Jam

**Permasalahan:**&#x20;

saat melakukan print kertas kurang menarik kertas sehingga hasil cetak pada kertas ketinggalan

**Indikasi:**

karet roller cekung dan licin

**Solusi:**

cuci karet roller dan melapisi dengan double tip

:white\_check\_mark: part bisa direpair mandiri

## Print Quality

**Permasalahan:**&#x20;

hasil cetak jelek, hampir tidak keluar sama sekali

**Indikasi:**

ink system tidak bisa menarik tinta dari print head saat power on maupun saat power off ke pembuangan dengan baik saat cleaning.&#x20;

karena printer lama tidak dinyalahkan nozzle printer juga sudah mulai mampet

**Solusi:**

ganti print head dan ink system

:white\_check\_mark: print head beli baru

:x: ink system belum tersedia
