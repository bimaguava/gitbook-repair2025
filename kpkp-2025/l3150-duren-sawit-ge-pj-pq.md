# L3150 Duren Sawit (GE, PJ, PQ)

## Paper Jam

**Permasalahan:**

kertas tidak tertarik

**Indikasi:**

karet roller cekung, gear dekat inksystem rontok

**Solusi:**

ganti roller dan gear

:white\_check\_mark: ambil part dari printer L3150 Kramat Jati

## General Error

**Permasalahan:**

Pergerakan carriage unit tidak bisa dikontrol mainboard

**Indikasi, pengecekan:**

compare encoder timing belt dan sudah cek jalur kabel encoder aman. lalu sisa sensor board di carriage unit yang harusnya rusak

**Solusi:**

ganti sensor board carriage unit

:white\_check\_mark: ambil part copotan (kanibal) printer L3150 Kramat Jati

## Print Quality

**Permasalahan:**

hasil cetak jelek

**Indikasi, pengecekan, dan repair:**

ink system masih belum bisa menarik tinta dengan baik saat cleaning. Disisi lain, print head masih jelek&#x20;

**Solusi:**&#x20;

jalur selang ink system direpair sehingga perputaran tinta bisa tertarik dengan baik ke pembuangan dan tidak mampet. Hasil nozzle print head hitam bagus, namun warna (cmy) tidak keluar, jadi ganti baru

:white\_check\_mark: ink system di repair

:white\_check\_mark: print head di ganti baru
